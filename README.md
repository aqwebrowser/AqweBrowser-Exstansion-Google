AqweBrowser

AqweBrowser is an open project focused on creating a clean, simple, and user-friendly browsing experience. Our goal is to reduce distractions and put users first with a fast, lightweight browser and extension.

🚧 The full browser is currently under development.

✅ The browser extension is available now and is free to use.

✨ Vision

The modern web is filled with advertisements, sponsored content, and AI-generated features that not every user wants. AqweBrowser aims to provide an alternative that focuses on simplicity, speed, and user choice.
